TERMINAL ! ! !

    Ishni boshlab olish uchun Unchalik murakkablashib ketmaymiz xozircha, Asosiylarini "TERMINAL" comandalarini Oddiy qilib Tushuntirib o'taman....

  • pwd

    Qayerda va qaysi papkada turganingizni ko'rsatadi

  • ls

    Turgan joyingizda "papkangizda" nimalar borligini ko'rsatadi

  • open .

    Siz turgan papkani alohida ochib ko'rish. Windowslarda (explorer .) bo'ladi

  • clear

    Terminaldagi yozuvlarni tozalash. Yoki control+k bossangiz xam bo'ladi

  • cd papka nomi

    Siz yozgan papkangizga kiradi

  • cd ~

    Homga qaytadi. Yani terminal ochganizda qaysi papkada tursa O'sha yerga qaytadi

  • cd ..

    Bitta orqaga qaytadi. Yani turgan papkangizdan orqadagi papkaga qaytadi

  • mkdir papka nomi

    Siz yozgan nom bilan papka yaratadi

  • code .

    Siz turgan papkani CodRedaktorda ochib beradi

  • rm -rf papka nomi

    Siz yozgan papkani O'chiradi!. Shuni yodda tutingki terminaldan o'chirgan narsangiz arxiv qutisiga tushmaydi!!! umuman o'chib ketadi

  • touch fayl nomi

    Siz yozgan nom bilan fayl yaratadi

  • rm fayl nomi

    Siz yozgan faylni O'chiradi

  • nano fayl nomi

    Siz yozgan Faylni TextRedaktorda ochib beradi

  • mv fayl yoki papka nomi va  yangiNom yoki QayergaKo'ckirish

    Papka yoki Faylni Nomini o'zgartirish Yoki boshqa joyga ko'chirish

  • cp fayl nomi va Qayerga Kopiya qilishni To'liq Ko'rsating

    Faqat fayllarni ko'piya qiladi Faylni tanlaysiz va Qayerga ko'piya qilishni aniq ko'rsatasiz. Masalan: /Users/mac/Desktop/portfolio





    GIT ! ! ! and GitHub ! ! !

  • git --version

    Sizning qurulmangizda GIT bor yo'qligini tekshirin. Agar bo'lsa ko'satadi

  • git config user.name

    gitdan foydalanuvchi ismini ko'rsatadi. Yani sizning githubdagi ismingizni

  • git config --global user.name "nikingizni yozing"

    GitHubdagi ismingizni yozing ! ! !

  • git config user.email

    Gitga bog'langan emailingizni ko'rsatadi

  • git config --global user.email "GitHubdagi emailingizni yozing "

    Gitni GitHub bilan bog'lash kerak. Demak GitHubdagi emailni yozishingiz shart ! ! !

  • git --help

    Gitning comandalarini chiqarib beradi Xammasini

  • git init

    Siz turgan papkaga Gitning ko'rinmas papkasini yaratadi

  • git status

    Sizning papkangizni holatini Ko'rsatadi. Yani Sizning Proektingizda GitPapkasiga Saqlanmagan fayllarni ko'rsatadi

  • git add .

    Sizning Proektingizdagi xamma O'zgarishlarni olib bir qutiga solib Gitga saqlashga tayyorlaydi desak xam bo'laveradi

  • git reset

    Add qilib qoshganingizni bekorqiladi

  • git reset fayl nomi

    Add qilib qoshganingizni bekorqiladi. Endi Faqat siz Yozgan faylni

  • git commit -m "izox yozasiz"

    Yuqoridagi qutini ustiga izox yozasiz. Va barcha o'zgarishlar Gitga Saqlanadi Shu izoh bilan siz uni Topishingiz Oson Bo'ladi

  • git commit -a -m "izox yozing"

    Bu bilan siz bir vaqtning o'zida Add Qo'shib va commit qilib Gitga saqlaysiz

  • git log

    va

    git reflog

    Sizning Gitga Jo'natgan Xamma commitlaringizni "IZOXLARIZNI" ko'rsatadi

  • git checkout branch nomi yoki commit manzilini kiriting

    Siz yozgan manzilga o'tadi

  • git checkout -- fayl nomini kiriting

    Siz yozgan fayldagi o'zgarishlarni bekor qiladi

  • git checkout -- .

    Barcha fayldagi o'zgarishlarni bekor qiladi

  • git branch

    Qaysi brenchda Turganizni Ko'rsatadi Agar boshqa branchlar bo'lsa ularni xam Ko'rsatadi

  • git branch Nom kiriting

    Yangi branch yaratadi

  • git branch -b Nom kiriting

    Yangi branch ochib Unga o'tadi

  • git branch -D branch nomini kiriting

    Siz yozgan branchni o'chiradi

  • git merge branch nomi

    Bu komonda faqat masterga Yani assosiy branchdagi Odamga boshqa branchni asosiyga qo'shish imkoniyatini beradi

  • git pull

    GitHubdagi Proektingizdagi O'zgarishlarni Olish

  • git push -u origin master

    GitHubdagi Proektingizga O'zgarishlarni Jo'natish
